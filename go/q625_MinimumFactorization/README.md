## Description
Given a positive integer a, find the smallest positive integer b whose multiplication of each digit equals to a.
If there is no answer or the answer is not fit in 32-bit signed integer, then return 0.

 
##### Example1:
Input:
48 

Output:
68

##### Example2:
Input:
15

Output:
35
